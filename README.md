# Alien Archives — Web App with Flask + Text-to-Speech

## Student: Maurice Weeks

---

## Project Description

This web application allows users to read and listen to fictional alien-themed eBooks. The system is fully offline and uses:

- Flask (Python backend web framework)
- pyttsx3 (Python text-to-speech engine)
- HTML, CSS (Frontend UI)
- REST API using JSON

This project demonstrates full-stack web development, REST API design, and backend-to-frontend integration.

---

## Technology Stack

- Python 3.x
- Flask
- pyttsx3 (offline TTS engine)
- HTML5
- CSS3
- JavaScript (Frontend TTS interaction)
