Welcome to **Alien Archives** — a simple interactive web application that showcases an alien-themed digital ebook library with integrated AI-powered text-to-speech functionality.

This project was built as part of my Web Applications & Databases course assignment.

---

## 🌌 Project Overview

Alien Archives features five unique sci-fi stories involving aliens, first contact, abductions, galactic wars, and hidden government secrets. Each book can be read and listened to using integrated text-to-speech (TTS) technology powered by a Flask-based backend.

---

## 🚀 Features

- 📚 **eBook Collection**  
  5 fully generated science fiction book titles with custom cover images.

- 🎙 **Text-to-Speech**  
  Full TTS support using `pyttsx3` and a Python Flask backend server.

- 🖥 **Web Application**  
  Simple responsive UI built with HTML, CSS, and vanilla JavaScript.

- 🌐 **GitHub Pages Deployment**  
  Hosted live on GitHub Pages.
